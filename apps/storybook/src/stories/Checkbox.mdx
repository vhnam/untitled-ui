## Overview

Checkboxes allow users to choose one or multiple values from a range of options. Use the Radio button component if the user should select only one value.

## Imports

```js
import { Checkbox } from "@repo/ui";
```

## Usage

Checkbox allow users to select one OR more options from a list.
Generally, selection controls allow users to complete tasks which involve making choices such as selecting options, switching settings on or off, or adding or removing items.
