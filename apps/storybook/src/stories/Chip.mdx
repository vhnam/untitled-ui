# Chip

- [Overview](#overview)

## Overview

Chip help people enter information, make selections, filter content, or trigger actions

## Imports

```js
import { Chip } from '@repo/ui';
```

## Usage

- Chips can be used to allow users to make single/multi selections (For e.g. selecting time-slots for an appointment)
- Chips can also be used to filter content; They can be grouped together to form horizontal carousel and are placed above / near content which users can filter using the chips.
- (Filter) chips can be used for single/multiple selections at a time
