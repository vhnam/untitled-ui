# IconButton

- [Overview](#overview)

## Overview

A type of button that does not have a visible label but instead an icon to denote its action. The component does require a label for accessibility & by default, will show a tooltip with that label.

## Usage

```js
import { IconButton } from "@repo/ui";
```
